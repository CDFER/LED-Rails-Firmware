Wi-Fi setup and feature updates for your Live Train Map.

  1. Plug your map into a USB port on your computer using the supplied USB Cable.
    Note: You can only use the USB port on the left side on Auckland maps for data transfer.
  2. Make sure you have selected the correct city and version of your map below.
    You can check the version of your map by looking for the label on the backside of the map.
  3. Click "Connect to Device" and select the correct COM port (USB JTAG/serial debug unit).
  4. Select "Connect to Wi-Fi" and follow the prompts (Installing the latest Firmware is optional).
Sorry, your browser does not support Web Serial. Please use Chrome or Edge. This tool requires HTTPS. Please use a secure connection.
